## Développer

- Ajouter une issue
- Créer une branche
- Ouvrir une MR vers la branche `master`
- Tester: lancer les tests généraux `npm test` et les tests clients `npm run test-client`

## Release

- Mettre à jour la date dans le CHANGELOG.
- `npm version <minor|patch>`
- `npm publish`
